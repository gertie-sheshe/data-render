DATA - RENDER
=============
React App that increments data after every 5 seconds and renders updated data

Installation.
-------------
1. Install [**Nodejs**](www.nodejs.org) 
2. Clone this repo or download the zipped file.
3. Navigate to the master branch.
4. Run `npm i`
   This will install the required dependencies.
5. Run `npm run dev`
   to run the app in development.
6. Run `npm run build`
   to run production ready app
7. Access the app on `localhost:8080`
